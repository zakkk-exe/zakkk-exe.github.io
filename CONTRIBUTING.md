# Contribuir a Shattered Coast Roulette

¡Gracias por tu interés en mejorar este proyecto! Al ser un fan-project simple, no hace falta mucha ceremonia.

## Cómo proponer cambios

1. Haz un fork del repositorio.
2. Crea una rama descriptiva: `git checkout -b actualiza-temporada-4`.
3. Haz tus cambios (ver abajo por dónde empezar según el tipo de cambio).
4. Prueba abriendo `index.html` en el navegador y girando cada rueda al menos una vez.
5. Abre un Pull Request explicando qué cambia y por qué.

## Dónde tocar según el tipo de cambio

| Quiero... | Archivo a editar |
|---|---|
| Actualizar POIs, armas, Sprites, movilidad o retos de una nueva temporada | `data.js` |
| Cambiar colores, tipografía o layout | `styles.css` |
| Cambiar el comportamiento de la ruleta (animación, lógica de giro, modo Run Completo) | `app.js` |
| Cambiar textos o estructura de la página | `index.html` |

## Estilo de código

- JavaScript vanilla, sin frameworks ni dependencias de build.
- Comentarios y textos de cara al usuario en español.
- Mantén los nombres de POIs, armas y Sprites fieles a los oficiales de Fortnite para que el contenido siga siendo preciso.

## Reportar un error

Abre un Issue describiendo:
- Qué esperabas que pasara.
- Qué pasó en realidad.
- Navegador y sistema operativo.
